Git and GitHub are used on all files within Gilboa_Water_Temp. After a file is modified:
Save changes to your local directory (your directory should be identical to the one above).
Stage all files.
Commit the files with a message.
Within GitHub, select the “Release” function.
Enter a tag name of the next release number (i.e. Release_4).
Enter a release name of the “release number – Receiver version number – Sender version number” (i.e.  “Release 4 - Receiver v1.3.02 - Sender v1.2.03”).
Attach the binaries (firmware.bin) for the Receiver and Sender. Renaming them with their corresponding target (Receiver_firmware.bin and Sender_firmware.bin).
